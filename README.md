# Creacion y suma de Matrices con Numeros Aleatorios en Python

Programa educativo con **3 ejercicios fundamentales** para aprender a trabajar con **matrices** (listas de listas) en Python usando números aleatorios.

### Qué hace exactamente este código

#### Ejercicio 1 – Matriz 3×3 aleatoria
- Genera una matriz 3×3 con números aleatorios del 1 al 10
- Muestra los números (aunque hay un pequeño detalle en el formato)
- Al final imprime la matriz completa como lista de listas

#### Ejercicio 2 – Suma de dos matrices
- Pide al usuario el tamaño `n` (crea matrices n×n)
- Genera dos matrices `A` y `B` con números aleatorios
- Calcula automáticamente `C = A + B` (suma elemento por elemento)
- Muestra las tres matrices

#### Ejercicio 3 – Suma y resta de matrices
- Igual que el ejercicio 2, pero además calcula:
  - `C = A + B` → suma
  - `D = A − B` → resta
- Muestra las cuatro matrices: A, B, C y D
